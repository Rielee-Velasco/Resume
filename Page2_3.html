H Drive Read and Write Access

A. Read Access – This is by default provided to users with H drive provisioning.

B. Write Access – After the OneDrive migration, write access is no longer default for users.

In the default setup, Home Drive has limited access permissions, specifically Read Only (R/O). It is not possible to store any data on Home Drive (H:), except for the folders that have been approved for Application data, which include System and Explorer/Favorites. These approved folders grant Read and Write (R/W) access.

Additional information can be found here: H: Drive Permissions

Examples of application folders are:

  • H:\DATA\SapLogon
  • H:\ESOPS

Request for access to H drive application folders can be placed here: H: Drive Permissions Request Form
